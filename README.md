# All Things Linear.

# About

Linear is a Psuedo 3D (2.5D) Game Engine Written entirely from scratch In Java zero external librarys.
It is fast optimized, open source (Under MIT). Linear Algebra is not used.

"How is that possible?". You'd be surprised that there is a lot you can do with simply just using basic
numbers and stuff to get a program to do what you want. All The "Math" is handled in the Math package.
Heck even rotation, Zero linear algebra or frustrating rotation matrix's just pure intellect that has all
the relevance to the engine's Workings.

"Why?". Part of it is because I don't know any Linear Algebra (its actually mostly that) and the other
part of it is to keep the Engine Optimized. Complex Mathematics can take up quite the compution power! Especially
if its doing it constantly. So without the stuff it greatly enhances performance.

I am not taking any contribuations just letting you know but feature requests and feedback is always
Welcomed :)
